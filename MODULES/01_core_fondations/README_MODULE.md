# Module 1 : Fondations Core

Maîtrisez les bases absolues de Python : syntaxe, variables, types, et logique de contrôle.

## Contenu du Module

| Chapitre | Titre | Objectif |
|----------|-------|----------|
| 01 | Premiers Pas | Installer Python, configurer VS Code, écrire votre premier script |
| 02 | Variables & Types | Manipuler les types de données fondamentaux |
| 03 | Opérateurs | Effectuer des calculs et comparisons |
| 04 | Contrôle de Flux | Prendre des décisions dans votre code |
| 05 | Boucles | Répéter des actions efficacement |
| 06 | Listes & Tuples | Stocker des séquences ordonnées |
| 07 | Dictionnaires & Sets | Utiliser des associations clé-valeur |

## Prérequis

Aucun. Ce module est le point de départ idéal pour débutants absolus.

## Durée Estimée

2-3 semaines à rythme modéré.

## Objectifs d'Apprentissage

À la fin de ce module, vous serez capable de :
- Écrire et exécuter des scripts Python autonomes
- Manipuler les types de données fondamentaux (int, float, str, bool)
- Utiliser les structures de contrôle (if, for, while)
- Travailler avec les collections (list, tuple, dict, set)
- Gérer les entrées/sorties utilisateur
