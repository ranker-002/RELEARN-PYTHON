# Module 2 : Fonctions & Programmation Orientée Objet

Apprenez à écrire du code réutilisable et à modéliser des objets du monde réel.

## Contenu du Module

| Chapitre | Titre | Objectif |
|----------|-------|----------|
| 08 | Fonctions | Créer des blocs de code réutilisables avec tous types d'arguments |
| 09 | Modules & Packages | Organiser et importer du code |
| 10 | Classes & Objets | Définir vos propres types de données |
| 11 | Héritage & Polymorphisme | Créer des relations entre classes |
| 12 | Propriétés & Méthodes Spéciales | Personnaliser le comportement des objets |

## Prérequis

Module 1 completado (fondations core).

## Durée Estimée

3-4 semaines.

## Objectifs d'Apprentissage

À la fin de ce module, vous serez capable de :
- Créer et utiliser des fonctions avec différents types de paramètres
- Organiser votre code en modules et packages
- Concevoir des classes avec encapsulation et héritage
- Utiliser les propriétés et méthodes spéciales (__init__, __str__, etc.)
- Appliquer les principes SOLID de base
