# Projet: Automatisation de Rapports

Automatisez la génération et l'envoi de rapports par email.

## Objectif

Appliquer automation (emails, fichiers, planification) dans un outil professionnel.

## Difficulté

**Avancé** - Durée estimée: 8-10 heures

## Prérequis

**Module 5 requis** : Chapitre 20

## Fonctionnalités Attendues

### Génération
- Templates Jinja2 pour格式化 HTML
- Graphiques avec matplotlib
- Tableaux de données

### Envoi
- SMTP avec pièce jointe
- Emails HTML formatés
- Configuration mult-compte

### Planification
- Exécution programmée (cron/schedule)
- Rapports journaliers/semestriels
- Logs d'exécution
