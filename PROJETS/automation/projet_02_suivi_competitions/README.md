# Projet: Suivi de Compétitions E-sport

Surveillez les sites de compétition et notifiez les changements.

## Objectif

Appliquer web scraping, APIs et notifications dans un projet temps réel.

## Difficulté

**Avancé** - Durée estimée: 10-12 heures

## Prérequis

**Module 5 requis** : Chapitre 20

## Fonctionnalités Attendues

### Surveillance
- Scraping périodique de sites (Liquipedia, Esports.gg)
- API monitoring (Toornament, Battlefy)
- Détection changements (scores, planning)

### Notifications
- Email lors de changements
- Desktop notifications (notify-send)
- Webhooks Discord/Slack

### Interface
- Dashboard CLI avec stats
- Configuration YAML
- Historique des changements
