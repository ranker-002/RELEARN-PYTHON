# Projet: Dashboard Analytique Interactif

Créez un dashboard web pour visualiser des données.

## Difficulté

**Expert** - Durée estimée: 15-20 heures

## Prérequis

**Module 5 requis** : Chapitres 22-23

## Fonctionnalités Attendues

### Backend
- API FastAPI pour servir les données
- Endpoints: /data, /stats, /export

### Frontend
- Dash/Streamlit ou Flask + Template
- Graphiques interactifs (Plotly)
- Filtres dynamiques

### Données
- Connection base SQL
- ou Import CSV/Excel
