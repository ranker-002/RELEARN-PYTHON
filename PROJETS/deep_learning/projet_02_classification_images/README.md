# Projet: Classification d'Images CNN

Créez un CNN pour classifier des images (CIFAR-10).

## Difficulté

**Expert** - Durée estimée: 30-40 heures

## Prérequis

**Module 5 requis** : Chapitre 26

## Fonctionnalités Attendues

### Architecture CNN
- Conv2D layers
- MaxPooling2D
- Flatten, Dense, Softmax

### Techniques
- Data augmentation
- Batch normalization
- Dropout
- Transfer learning (optionnel)

### Dataset
- CIFAR-10 (10 classes d'images)
- Ou Flowers-102 (plus difficile)

### Résultats
- Accuracy > 70% sur CIFAR-10
- Visualisation feature maps
