# Projet: Modèle de Prédiction Immobilière

Construisez un modèle ML pour prédire les prix immobiliers.

## Difficulté

**Expert** - Durée estimée: 20-25 heures

## Prérequis

**Module 5 requis** : Chapitre 25

## Fonctionnalités Attendues

### Données
- Dataset Kaggle (ex: House Prices)

### Prétraitement
- Gestion valeurs manquantes
- Encodage variables catégorielles
- Feature engineering

### Modélisation
- Régression linéaire, Ridge, Lasso
- Random Forest
- Gradient Boosting (XGBoost, LightGBM)

### Évaluation
- Cross-validation
- Métriques (RMSE, MAE, R²)
- Feature importance
