# Projet: Classification Multi-classes

Créez un modèle de classification sur données tabulaires.

## Difficulté

**Expert** - Durée estimée: 20-25 heures

## Prérequis

**Module 5 requis** : Chapitre 25

## Fonctionnalités Attendues

### Problématique
- Classification binaire ou multi-classes
- Exemple: Prédiction churn, classification diseases

### Approches
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting
- SVM

### Optimisation
- GridSearchCV / RandomizedSearchCV
- Équilibrage des classes (SMOTE)

### Métriques
- Accuracy, Precision, Recall, F1
- ROC-AUC
- Matrice de confusion
