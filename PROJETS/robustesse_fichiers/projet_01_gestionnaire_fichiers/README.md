# Projet: Gestionnaire de Fichiers CLI

Créez un explorateur de fichiers sécurisé en ligne de commande.

## Objectif

Maîtriser la manipulation de fichiers, les exceptions, et pathlib dans un outil pratique.

## Difficulté

**Intermédiaire** - Durée estimée: 4-6 heures

## Prérequis

**Module 3 requis** : Chapitres 14-15

## Fonctionnalités Attendues

### Navigation
- Lister fichiers/dossiers avec détails (taille, date, permissions)
- Changer de dossier courant
- Naviguer vers le parent/racine

### Opérations
- Copier fichiers et dossiers
- Déplacer/renommer
- Créer dossiers
- Supprimer avec confirmation
- Rechercher par nom/motif

### Sécurité
- Validation des chemins (sortie du dossier racine interdite)
- Confirmation avant suppression
- Gestion gracieuse des erreurs
