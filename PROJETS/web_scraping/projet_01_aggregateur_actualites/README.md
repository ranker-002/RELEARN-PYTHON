# Projet: Agrégateur d'Actualités RSS

Créez un agrégateur de flux d'actualités avec filtrage intelligent.

## Objectif

Appliquer web scraping, parsing XML/JSON dans une application d'agrégation.

## Difficulté

**Avancé** - Durée estimée: 8-10 heures

## Prérequis

**Module 5 requis** : Chapitre 21

## Fonctionnalités Attendues

### Agrégation
- Parsing RSS/Atom feeds
- Support multi-sources configurables
- Filtrage par mots-clés/catégories

### Interface
- Affichage CLI paginé
- Ouverture articles dans navigateur
- Sauvegarder pour lecture ultérieure

### Bonus
- Recherche full-text
- Export OPML (transfert abonnements)
- Cache articles
