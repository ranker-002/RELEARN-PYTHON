# Projet: Analyseur de Prix E-commerce

Surveillez les prix sur plusieurs sites e-commerce.

## Difficulté

**Avancé** - Durée estimée: 10-12 heures

## Prérequis

**Module 5 requis** : Chapitre 21

## Fonctionnalités Attendues

### Collecte
- Scraping Amazon, Cdiscount, Fnac...
- Rotation user-agents
- Rate limiting respectueux

### Analyse
- Historique des prix
- Détection promotions
- Alertes seuil de prix

### Dashboard
- Graphique évolution prix
- Meilleures offres
- Export CSV/Excel
